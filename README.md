A touch-screen control panel for [FluidNC](https://github.com/bdring/FluidNC)
CNC controllers. Mount it on the side of your CNC machine and use it to
home the spindle, jog the axes, start a job, set work coordinates, run macros,
and watch what the machine is doing — all without reaching for a laptop.

**Hardware:** [Waveshare ESP32-P4 WiFi6 Touch LCD 7B](https://www.waveshare.com/wiki/ESP32-P4-WIFI6-Touch-LCD-7B) — 7-inch 1024×600 capacitive touch panel with an ESP32-P4 main chip and an ESP32-C6 WiFi co-processor.
**Firmware:** ESP-IDF 5.5.x · LVGL 8.4 · UI authored in [EEZ Studio](https://www.envox.eu/studio/).

---

## Quick start

1. Flash the firmware — see [docs/06-building-and-flashing.md](docs/06-building-and-flashing.md).
2. On first boot the pendant scans for WiFi networks — pick yours, enter the password.
3. Open Settings → Machine → "Configure FluidNC Connection" — type your
   controller's hostname or IP, pick **WebSocket** for wireless or **UART** for wired.
4. Tap **Connect** and the pendant lands on the Dashboard with live machine readouts.

## Project layout

| Path | Purpose |
|---|---|
| [main/](main/) | Application code — state machine, action handlers, variable bridge |
| [main/ui/](main/ui/) | UI generated by EEZ Studio — **never edit by hand** |
| [GUI/](GUI/) | EEZ Studio project file (`.eez-project`) — source of truth for the UI |
| [components/fluidnc/](components/fluidnc/) | FluidNC protocol — WebSocket / UART transports, grbl parser, dispatcher |
| [components/wifi_setup/](components/wifi_setup/) | WiFi scan + connect via the ESP32-C6 co-processor |
| [components/pendant_config/](components/pendant_config/) | Persistent settings stored in NVS (flash) |
| [assets/](assets/) | Source images, fonts, icons referenced by the EEZ Studio project |
| `sdkconfig.defaults` | Pinned project config — edit here, never `sdkconfig` |
